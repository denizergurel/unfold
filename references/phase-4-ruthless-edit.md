# Phase 4 — Ruthless edit: cut to only what matters

This is the phase that makes or breaks the artifact, and the one people resist
most. The art of writing is thrift. A natural, heartfelt brain-dump runs
10–15 minutes. The target is roughly **3–5 minutes**. That gap is closed only by
deliberate, repeated cutting.

## Expect to cut more than once

The first cut will still be too long. So will the second. Plan for several
passes. Each pass, read the whole thing fresh and ask what can go — not which
*words* to trim, but which *sections* to remove entirely. Trimming words gets
you 10%; cutting sections gets you to length.

## The cut checklist

For every section / slide, ask:

- **Does this demonstrate judgment or taste?** If it only lists an output,
  it's a candidate to cut.
- **Would the story survive without it?** If yes, cut it. The story should
  contain only load-bearing beats.
- **Is this here because it's impressive, or because it's *me*?** Impressive-
  but-impersonal is exactly what a resume already does. Keep the beats that show
  the person.
- **Am I explaining when I could show?** A real visual + one sentence often
  replaces a paragraph of narration.
- **Is there a shorter way to say this exact thing?** Tighten the survivors.

## The emotional trap

Cutting feels like loss — every beat carries a memory the person is fond of.
Name this for the user, then cut anyway. Fondness is not the same as
load-bearing. The goal is the strongest five minutes, not a complete record.

## Calibrating length

Read the script aloud (or have the user do it) and time it. Repeat the cut →
time → cut loop until it lands in the target window and still feels true to the
work. "Concise enough to hold attention, thorough enough to be honest" is the
bar.

## Output of this phase

A final narration script at target length, every surviving beat justified by the
checklist above. Record what you cut, and why, in the cut log of
`assets/script-skeleton-template.md` — it makes re-tailoring (Phase 5) easier.

## Anti-patterns (don't do this)

- **Trimming words instead of cutting sections.** Word-level edits get ~10%. To
  hit length you must remove whole beats. Cut sections first.
- **Keeping a beat because it's fond, not because it's load-bearing.** Sentiment
  is not a reason to keep a section. Name the fondness, then cut.
- **Declaring the first cut final.** It will still be too long. Plan multiple
  passes; read aloud and time each one.
